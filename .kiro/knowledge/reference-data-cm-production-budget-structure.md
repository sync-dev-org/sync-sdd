## Overview

**CM制作会社における実行予算の構成・費目分類**

広告映像制作プロダクションの予算管理は2段階構造: (1) 見積書は受注前の概算提示で顧客向け、(2) 実行予算書は受注後の詳細管理で社内向け。費目分類は企画費・撮影費・編集費・人件費・諸経費の5-6カテゴリが一般的だが、業界標準は未確立。

## Quick Reference

### 予算構成の基本

| 項目 | 説明 |
|------|------|
| 見積額 | 企画費 + 人件費 + 諸経費の3大分類 |
| 積算方式 | 人件費(スタッフ別単価) x 拘束日数 + 機材費(機材別単価) x 使用日数 + 諸経費 |
| 見積書 | 受注前の概算・顧客提示用（契約前作成） |
| 実行予算書 | 受注後の詳細・内部管理用（受注後作成） |
| 詳細度の差 | 見積書は概算項目、実行予算書は材料・工程ごとの詳細項目 |

### 費目6分類と相場

| 費目 | 相場 |
|------|------|
| 企画費 | 2-50万円（企画構成案5-50万円、絵コンテ5-30万円、ディレクション費10-100万円以上） |
| 制作費 | 5-100万円 |
| 撮影費 | 15-100万円（機材1-20万円、撮影スタッフ5-20万円/日、キャスト5-30万円、スタジオ10-30万円） |
| 編集費 | 0.5-3万円基本（グラフィック0.5-1万円/枚、ナレーション3-10万円、音響3-15万円、編集10-50万円、MA5-10万円） |
| 人件費 | 5-300万円 |
| 諸経費 | 案件による |

## Key Points

### 日本の予算管理
- 管理粒度は「案件 x 工程 x スタッフ x 日数」レベルでの原価把握が重要
- 「時間 x 人 + 外注費」の構造で損益を判断
- 費目確認項目: 交通費含有、著作権帰属を必ず確認

### 米国の制作予算体系
- Pre-production（脚本・キャスティング・ロケハン）、Production（撮影日経費）、Post-production（編集・音楽・VFX）の3段階
- 総予算の10%を予備費（contingency）として確保が業界標準
- 人件費比率: 総予算の30-40%

## Common Gotchas

| Gotcha | Impact | Mitigation |
|--------|--------|------------|
| 業界標準フォーマット未確立 | 各社でフォーマットや項目名の表記が異なる | 自社フォーマットを定義し一貫して使用 |
| 映像制作専用の予算管理ソフトが限定的 | 建設業向けツールが多く映像制作に特化したものは少ない | 汎用ツールをカスタマイズ or 内製 |
| 日米の用語・構造差異 | 米国の制作予算体系と日本で用語・比率が異なる | 日本のクライアント向けには日本式で統一 |
